# Orders Vs Returns by Customer and Product Analysis

## Project Overview
This project analyzes customer orders and product returns to identify high-return products and customers using Azure Databricks and PySpark.

## Objective
Get the Top 5 products and customers with highest orders and returns to identify potential issues and improve customer satisfaction.

## Technologies Used
- Python
- PySpark
- Azure Databricks

## Architecture
Bronze Layer → Silver Layer → Gold Layer

## Implementation Steps
1. Ingested data using Auto Loader
2. Cleaned and standardized data
3. Stored transformed data in Silver layer
4. Built KPI tables in Gold layer
5. Calculated return percentage
6. Identified top customers and products

## KPI Logic
- Top 5 products and customers with highest orders

- PySpark notebook
- Python script
